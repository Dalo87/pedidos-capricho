🔥 Configura Firebase

Para que el comercial y administración vean los pedidos en tiempo real, necesitas conectar Firebase (gratis). Solo se hace una vez.

  1. Ve a console.firebase.google.com → Crear proyecto
  2. Activar Realtime Database → Crear base de datos → Modo de prueba
  3. Proyecto → Configuración → Tu aplicación web → Registrar app
  4. Copia las credenciales y pégalas en el código (busca TU_API_KEY)
